# Hungry-Mobile-end
这是在学习的过程中制作的一个模仿饿了么点餐的移动端作品，可实现短信接收、微信支付、地图定位等的功能！
短信的接口因为我是个人开发者所以我用的是云之讯平台的接口，地图部分用的是高德地图的地图JS Api和Key。因为大部分电脑上没有GPS芯片，所以用的是ID地址定位，如果使用代理的话可能会导致偏差。
